# 19. 의료 API 응답 예시

## 예시 1. 반복되는 위험 신호

질문:
- 환자 추적 관찰에서 반복되는 위험 신호는 무엇인가?

예시 응답 포인트:
- 복약 순응도 저하
- 추적 관찰 지연
- 만성질환 중복 관리 필요성
- 고위험군 우선 추적 필요

## 예시 2. 복약 순응도 문제

질문:
- 복약 순응도와 관련해 자주 보이는 문제는 무엇인가?

예시 응답 포인트:
- 복약 누락
- 저녁 복약 누락 패턴
- 혈압약/당뇨 관련 약 복용 불규칙
- 모니터링 기록 누락과 연결되는 경우

## 예시 3. follow-up 지연

질문:
- follow-up 지연이 있는 환자들의 공통점은 무엇인가?

예시 응답 포인트:
- 고위험군 비율이 높음
- 만성질환 동반 비율이 높음
- 복약 이슈와 함께 나타남
- 재방문 지연이 관리 불안정성으로 이어질 수 있음

## 예시 4. 만성질환 공통 관리 문제

질문:
- 당뇨와 고혈압 환자에서 공통적으로 보이는 관리 문제는 무엇인가?

예시 응답 포인트:
- 복약 순응도 저하
- 자가 모니터링 불규칙
- follow-up 지연
- 지속적 생활 관리 필요
