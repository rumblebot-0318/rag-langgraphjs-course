# 17. 의료 샘플 질문 예시 세트

이 문서는 sample medical dataset으로 바로 테스트해볼 수 있는 질문 예시를 모아둔 것이다.

## 기본 질문

- 환자 추적 관찰에서 반복되는 위험 신호는 무엇인가?
- 복약 순응도와 관련해 자주 보이는 문제는 무엇인가?
- follow-up 지연이 있는 환자들의 공통점은 무엇인가?
- 고위험 환자에서 반복되는 관리 이슈는 무엇인가?

## 만성질환 중심 질문

- 당뇨와 고혈압 환자에서 공통적으로 보이는 관리 문제는 무엇인가?
- 다중 만성질환 환자에서 더 자주 보이는 위험 패턴은 무엇인가?
- 혈당 모니터링과 혈압 관리 측면에서 어떤 문제가 반복되는가?

## 운영/관리 질문

- 추적 관찰이 지연될 때 어떤 위험이 커질 수 있는가?
- 환자 기록에서 follow-up 우선순위를 높여야 할 사례는 어떤 유형인가?
- 고위험군 환자를 먼저 관리하려면 어떤 기준을 볼 수 있는가?

## API 테스트용 질문 예시

```json
{
  "question": "환자 추적 관찰에서 반복되는 위험 신호는 무엇인가?",
  "topK": 4,
  "provider": "ollama",
  "model": "qwen2.5:3b",
  "temperature": 0.2
}
```
