# data

이 폴더는 실습 데이터를 넣는 위치다.

- `sample.db` : SQLite 샘플 DB
- `sample-report.txt` : 기본 문서형 샘플 데이터
- `sample-report.md` : 같은 내용을 markdown으로 정리한 참고본
- `sample-report.pdf` : 선택적으로 PDF 파싱을 시험해보고 싶을 때 쓰는 보조 파일

기본 데모는 `sample-report.txt`와 `sample.db`를 함께 읽는다.
즉, 텍스트 요약과 DB row를 같이 검색 대상으로 넣는 흐름을 바로 설명할 수 있다.
